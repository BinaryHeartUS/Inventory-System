The following are the meanings behind digits in file names:

1st digit: NYI
2nd digit: NYI
3rd digit:

- 1: Creating types
- 2: Creating tables
- 3: Creating stored procedures
- 4: Other scripts that need to run
  4th digit: Indicates order within level (e.g. something with the 4th digit '2' is depending on something with the 4th digit '1' or '0')
  5th digit: Separation for files within the level
